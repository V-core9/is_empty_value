# 👨‍💻 is_empty_value

🔩 Install using command :

    npm install is_empty_value

🎭 And save for later re-install :

    npm install is_empty_value --save

🪁 How to use:

    const is_empty_value = require("is_empty_value");

    console.log( is_empty_value("demo_password_123456") )
